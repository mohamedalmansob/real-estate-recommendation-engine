# Real Estate Recommendation Engine

## Overview

A smart location-aware real estate recommendation system that recommends suitable properties based on user location and desired criteria using geospatial analysis and intelligent scoring techniques.

## Project Goal

The goal of this project is to build a recommendation engine that ranks real estate properties according to:

- Geographic proximity
- Property characteristics
- User requirements
- Nearby services and facilities

The system aims to help users find the most suitable properties without requiring historical user preferences.

## Dataset

The project uses real estate market data containing information about properties, including:

- Price
- Bedrooms
- Bathrooms
- Living area
- Property type
- Construction year
- Geographic coordinates (latitude and longitude)
